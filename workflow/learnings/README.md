# Learnings

Learnings capture process improvements discovered during Reflect.

## Files

| Path | Purpose |
|---|---|
| `sessions/` | Raw session entries written by Reflect. |
| `curated.md` | Human-approved durable learnings. |

## Rules

- Reflect writes raw learning sessions.
- Raw sessions are append-only.
- Candidate learnings must be tagged `propose-only`.
- Do not edit `curated.md` unless the user explicitly requests curation.
- Learning candidates should cite the artifact or evidence that produced them.
